Serviços

Os serviços oferecidos são o desenvolvimento de produtos ou serviço por meio da Experiência do Usuário e do Design de Interface(UX/UI). O método utilizado é o do Design Thinking e etapas e suas entregas podem variar de acordo com a necessidade observada.

Os serviços de Web Design são o desenvolvimentos de sites por meio de HTML, CSS, JavaScript e WordPress. Dependendo da necessidade, o médo pode muda. Minha principal especialidade são sites institucionais.

UX/UI

Etapas e Entregas

• Briefing com o cliente

Conversa com o cliente para fazer um levantamento das demandas do cliente, onde serão estudadas datas, etapas e entregas.

• Empatia

A primeira fase do Design Thinking é a Empatia, que é o processo de entender o usuário e suas necessidades. Nesta fase, a equipe deve realizar pesquisas e entrevistas com usuários. Observar seus comportamentos e identificar suas necessidades, desejos e desafios. Esta fase é crucial para a compreensão do problema. E para a criação de soluções que atendam às necessidades reais do usuário.

• Definição do problema

Após a fase de Empatia, a equipe deve definir o problema a ser resolvido.

• Ideação

A terceira fase do Design Thinking é a Ideação, que é o processo de gerar ideias para resolver o problema.

• Prototipagem

O objetivo da fase de Prototipagem é testar as soluções para identificar possíveis problemas e refiná-las antes da implementação.

• Teste

Processo de testar os protótipos com os usuários reais dentro do público alvo e obter feedback para aprimorar a solução.

  • *Entregas podem variar de acordo com a necessidade do cliente
  • *Uma etapa só seguirá para outra, se o cliente aprovar.

Web Design

Etapas e Entregas

• Briefing com o cliente

Conversa com o cliente para fazer um levantamento das demandas do cliente, onde serão estudadas datas, etapas e entregas.

• Desenvolvimento do protótipo

Caso o cliente já tiver um produto bem desenvolvido em termos de UX, é possível ir diretamente para a fase de prototipação, onde será criado um protótipo do site para depois, começar as fases de codificação. Caso o cliente já tenha o protótipo, podemos pular diretamente para o Desenvolvimento do código.

• Desenvolvimento do código

Tendo o protótipo em mãos, aprovado pelo cliente, o site começará a ser desenvolvido em HTML, CSS. e em JavaScript.

Com relação a Hospedagem e Domínio, será feita uma cotação de qual seria o mais vantajoso. O valor será incluído no orçamento.

  • *Uma etapa só seguirá para outra, se o cliente aprovar.
